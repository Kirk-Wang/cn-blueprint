Core

Core

@blueprintjs/core NPM包是任何Blueprint应用程序的基础。它包括许多（30+）React组件，覆盖所有基本的基础组件，从按钮到表单控件到工具提示和树。它还包括每个组件的CSS样式以及使用Sass和Less变量设计自己的组件和应用程序的工具，优雅的调色板以及两种尺寸的300多个UI图标。
